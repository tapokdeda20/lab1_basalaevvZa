# Lab1_FIO

Консольное приложение на C# для определения типа треугольника и расчета координат его вершин.

## Возможности
- Определение типа треугольника
- Проверка корректности входных данных
- Расчет координат вершин
- Масштабирование в поле 100x100
- Логирование в файл и консоль

## Входные данные
Три стороны треугольника 

## Выходные данные
- Тип треугольника
- Координаты вершин

## Логирование
Все запросы записываются в log.txt
